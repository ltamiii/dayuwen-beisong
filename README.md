# 大学语文背诵默写题生成器

一个零依赖的单文件网页工具，辅助背诵大学语文 8 篇指定篇目。支持**学习模式**（注音、注释、逐句翻译、背景中心思想）和**默写模式**（三级难度随机挖空，自动生成模拟默写题）。

## 快速部署

### GitHub Pages（推荐）

1. 在 GitHub 新建仓库，将 `index.html` 上传到仓库根目录
2. Settings → Pages → Source 选 `main` 分支，点 Save
3. 几分钟后访问 `https://你的用户名.github.io/仓库名/`

### 任意静态托管

把 `index.html` 放到任意静态文件服务器即可，无需构建、无需安装依赖。也支持：
- **Netlify Drop**：直接拖拽 `index.html` 到 [app.netlify.com/drop](https://app.netlify.com/drop)
- **Vercel**：`npx vercel` 一键部署
- **本地预览**：`npx serve .` 然后浏览器打开

## 使用说明

### 学习模式（默认）

| 功能 | 操作 |
|------|------|
| 切换篇目 | 左侧边栏点击课文标题 |
| 查看注音 | 原文中生僻字上方自动标注拼音 |
| 查看注释 | 点击带**虚线**的词语，右侧注释栏自动滚动并高亮 |
| 查看逐句翻译 | 点击带**点状下划线**的句子，下方展开现代汉语译文；再点收起 |

### 默写模式

| 功能 | 操作 |
|------|------|
| 切换模式 | 左侧边栏点击「✏️ 默写」 |
| 选择难度 | 简单(~35%) / 中等(~50%) / 困难(~70%) |
| 重新生成 | 点击「🔄 重新生成」或按键盘 `R` 键 |
| 显示/隐藏答案 | 点击「👁 显示答案」，答案以红色显示；再点隐藏 |
| 标点保留 | 挖空只替换汉字，逗号句号等标点原样保留，自然分隔 |

### 快捷键

| 按键 | 作用 |
|------|------|
| `R` | 默写模式下重新生成题目 |

## 技术说明

- 纯静态 HTML，CSS/JS 全部内嵌，无外部依赖
- 所有课文数据以 JavaScript 对象存储在页面中
- 挖空算法使用确定性伪随机（每次生成不同组合，保留标点符号）
- 响应式布局，支持桌面端和移动端
